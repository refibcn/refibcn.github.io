§2 Color — pick & tune

Broad, flexible palette for many surfaces and complex data — still recognizably ReFi BCN. Your canvas brief structures it in three layers:

  • Paper neutrals — soft backgrounds and text
  • Three pillar ranges — four steps each for Neural Networks, Tissues, and Flows
  • Color library — extra hues for UI and data visualizations
Use the color inputs below; changes auto-save and apply on all preview pages after refresh. Copy blocks for tokens.css when ready.

Palette references

Inspiration gathered as a whole — not a menu of mutually exclusive options. Pull from these when tuning pillars and the library below. Source images are optional (hidden by default).

§2.1 Paper neutrals

Tune with the pickers below — saves to this browser only. When happy, Download colors-locked.json + run the apply script (see footer), or use Copy paper block.

§2.2 Pillar color ranges

Three pillars side by side — four editable steps each (lightest → deepest) plus a vertical gradient preview. Tune against the palette references above. Picks save to this browser only until you export.

§2.3 Color library

Six steps per hue (Tint → Shade). Only Core (step 4) is editable and maps to tokens.css; lighter and darker stops are mixed from each family’s tint and shade anchors. Tune against the palette references above — not locked to any single source.
